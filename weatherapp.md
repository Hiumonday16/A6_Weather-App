# Weather App - Portfolio Overview

## Project Description
A responsive, interactive weather application that provides real-time weather information for cities worldwide. Built with vanilla JavaScript, HTML, and CSS, this app demonstrates proficiency in API integration, asynchronous programming, and dynamic UI updates.

## Key Features

### Core Functionality
- **City Search**: Search for weather information by entering any city name
- **Real-time Weather Data**: Fetches current weather conditions using the OpenWeatherMap API
- **Comprehensive Weather Display**: Shows temperature, weather conditions, visibility, wind speed, and humidity
- **Local Time Display**: Automatically calculates and displays the local time for the searched city based on timezone data

### Dynamic User Experience
- **Adaptive Backgrounds**: Background images dynamically change based on temperature ranges:
  - Hot (>25°C): Warm, sunny imagery
  - Cool (15-25°C): Moderate weather scenes
  - Warm (15-20°C): Pleasant weather visuals
  - Cold (≤15°C): Cool, winter-themed backgrounds
- **Smooth Transitions**: CSS transitions provide fluid visual feedback during interactions
- **Error Handling**: Gracefully handles invalid city searches by hiding content

### Technical Implementation
- **API Integration**: Seamless integration with OpenWeatherMap REST API
- **Asynchronous Programming**: Utilizes async/await for efficient API calls
- **Temperature Conversion**: Converts Kelvin to Celsius for user-friendly display
- **Event-Driven Architecture**: Search functionality triggered by Enter key press
- **Responsive Design**: Centered card layout that works across different screen sizes

### UI/UX Highlights
- **Modern Design**: Clean, minimalist interface with glassmorphism-style elements
- **Visual Feedback**: Interactive search input with focus states and smooth transitions
- **Icon Integration**: Font Awesome icons for enhanced visual communication
- **Typography**: Clear hierarchy with text shadows for readability over dynamic backgrounds

## Technologies Used
- **HTML5**: Semantic markup structure
- **CSS3**: Advanced styling with gradients, transitions, and responsive design
- **Vanilla JavaScript**: No framework dependencies, demonstrating core JavaScript skills
- **OpenWeatherMap API**: External API for weather data retrieval
- **Font Awesome**: Icon library for visual elements

## Skills Demonstrated
- RESTful API integration and data handling
- Asynchronous JavaScript (async/await, fetch API)
- DOM manipulation and dynamic content updates
- CSS animations and transitions
- Responsive web design principles
- Error handling and user experience optimization
- Temperature unit conversion and timezone calculations
